📊 Business Analytics Dashboard System

🚀 Overview

A full-stack Spring Boot Business Analytics Dashboard System designed for real-time sales tracking, product management, and inventory insights.

This project helps businesses manage products, monitor sales, and analyze performance using interactive dashboards.

✨ Key Features
📊 Dashboard Analytics
Total Products, Sales, Revenue
Today’s Sales tracking
Best-selling product detection
Interactive charts (Chart.js)

📦 Product Management
Add / Edit / Delete products
Stock status:
🟢 OK
🟡 LOW
🔴 OUT OF STOCK

💰 Sales Management
Add sales with automatic calculation
Stock auto-reduction
Sales history tracking

⚠ Low Stock Alerts
Critical / Warning / Safe levels
Sorted low-stock products
Real-time monitoring

🗑 Soft Delete System
Move sales to Trash Bin
Restore deleted records
Permanent delete option

🛠 Tech Stack
Java
Spring Boot
Spring Data JPA
Thymeleaf
MySQL
HTML, CSS
Chart.js
